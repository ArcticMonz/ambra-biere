# 🍻 Projet Birra Italo-Belga

Ceci est un projet universitaire consistant en un site web statique pour une brasserie artisanale combinant les traditions italiennes et belges.

## 📖 Description

Le site présente une brasserie fictive, son histoire et son catalogue de bières. Le projet se concentre sur la structure sémantique et le design visuel.

## ✨ Fonctionnalités

* **Accueil :** Présentation de la brasserie.
* **Catalogue :** Liste des bières disponibles.
* **Histoire :** La fusion entre le savoir-faire italien et belge.
* **Contact :** Formulaire de contact (statique).

## 🛠️ Technologies utilisées

* **HTML5** : Structure du contenu.
* **CSS3** : Mise en forme et design.
